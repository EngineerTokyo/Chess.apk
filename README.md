

## Project Description
This is an Android Chess Game developed in Android Studio. The game allows two players to play chess on a mobile device following standard chess rules. The project is developed as part of an academic assignment.

## Features
- Two-player chess game
- Simple and user-friendly interface
- Valid chess moves
- Reset game option

## Technologies Used
- Android Studio
- Java / Kotlin
- XML for UI design

## How to Run the Project
1. Clone or download the project from GitHub
2. Open the project in Android Studio
3. Connect an Android device or use Emulator
4. Click Run button
